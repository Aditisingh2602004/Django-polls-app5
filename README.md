Django Polls Application 
This is a Django-based Polls application developed as part of learning Django fundamentals and automated testing.  
The project is based on the official Django documentation and demonstrates clean project structure, views, templates, URL routing, and automated tests.

Project Overview
The Polls application allows users to:
- View a list of poll questions
- Vote on available choices
- See real-time voting results
- Prevent future-dated questions from being visible

The project also includes automated tests to ensure correctness and prevent bugs.

Tech Stack
- Python
- Django
- SQLite (development database)
- Git & GitHub
